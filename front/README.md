 # 사용법
  ### yarn 다운
   npm install -g yarn or brew install yarn
  ### berry mode 적용
   yarn set version berry
  ### pnp mode 활성화
  yarn add @yarnpkg/plugin-pnp
  yarn plugin import @yarnpkg/plugin-pnp
  ### vscode 사용시
  - 확장프로그램 : ZipFS
  - 필수 실행 : yarn dlx @yarnpkg/sdks vscode
